# CODSOFT Artificial Intelligence Internship Tasks

This repository contains the tasks completed as part of the **CODSOFT Artificial Intelligence Internship**.

---

## Task 1: Rule-Based Chatbot

### Description
A simple customer support chatbot that responds to user queries using predefined rules and regular expressions.

### Features
- Product information
- Technical support
- Return policy assistance
- General query handling
- Exit commands (bye, exit, quit, goodbye)

### Technologies Used
- Python
- Regular Expressions (re)

### Run

```bash
python Task_1_Chatbot_with_Rule_Based_Responses.py
```

# Task 3: Image Captioning

### Description

An image captioning application that uses a pre-trained transformer-based model (BLIP) to generate captions for input images.

### Features

- Image caption generation
- Pre-trained Transformer Model
- Automatic caption prediction

### Technologies Used

- Python
- Transformers
- PyTorch
- Pillow

### Install Dependencies

```bash
pip install transformers torch pillow
```

### Run

```bash
python Task_3_Image_Captioning.py
```

# Task 5: Face Detection

### Description

A face detection application using OpenCV's Haar Cascade Classifier.

### Features

- Detects faces in an image
- Draws bounding boxes around detected faces
- Displays total number of faces detected

### Technologies Used

- Python
- OpenCV

### Install Dependency

```bash
pip install opencv-python
```

### Run

```bash
python Task_5_Face_Detection.py
```
